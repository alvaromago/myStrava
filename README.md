# Nombre del proyecto

myStrava

## Descripción

Sencilla web que permite crear, mostrar y eliminar actividades.
Las actividades se guardan usando localstorage.

## Tecnologías

- HTML
- CSS
- JavaScript vanilla

## Funcionalidades

- Añadir actividades con nombre, deporte y fecha.
- Eliminar actividades existentes
- Guardado en localstorage para que los datos se mantengan al refrescar la página.

## Cómo usar

1. Abrir `index.html` en un navegador
2. Rellenar el formulario y hacer clic en "Añadir"
3. Ver las actividades en la lista
4. Hacer clic en "Eliminar" para eliminar una actividad
